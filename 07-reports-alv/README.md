# 07 — Reports & ALV

> ALV (ABAP List Viewer) é o padrão de exibição de dados em relatórios SAP. Saber construir um ALV decente — com ordenação, filtros, export Excel e eventos customizados — é habilidade obrigatória para qualquer desenvolvedor ABAP.

## Aulas

- [01 — ALV Grid com CL_SALV_TABLE](./01-alv-grid.md)
- [02 — ALV com Eventos e Toolbar](./02-alv-eventos.md)

## O que você vai aprender

| Conceito | Por que importa |
|----------|----------------|
| CL_SALV_TABLE | ALV OO moderno — menos código que o clássico |
| CL_GUI_ALV_GRID | ALV com controle total (eventos, toolbar customizada) |
| Double-click e hotspot | Navegação para detalhe |
| Export Excel | Recurso que todo usuário pede |
| Totalizadores e semáforos | Dados visuais e sumários |
