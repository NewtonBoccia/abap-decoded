# 09 — Fiori & RAP

> Fiori é o framework de UI moderno da SAP. RAP (Restful ABAP Programming Model) é o modelo de desenvolvimento para criar apps Fiori a partir de CDS Views + ABAP OO. É o futuro do desenvolvimento SAP — e o que diferencia um dev ABAP tradicional de um dev ABAP moderno.

## Aulas

- [01 — Fiori & RAP: Conceitos e Primeiro App](./01-fiori-rap-intro.md)

## O que você vai aprender

| Conceito | Por que importa |
|----------|----------------|
| CDS Views com anotações Fiori | Base de qualquer app RAP |
| Behavior Definition (BDEF) | Define operações: Create, Update, Delete |
| Behavior Implementation (ABAP OO) | Lógica de negócio das operações |
| Service Definition + Binding | Publica como OData para o Fiori |
| Fiori Elements | UI gerada automaticamente — sem código frontend |
