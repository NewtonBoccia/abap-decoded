# 03 — OO ABAP

> Orientação a objetos em ABAP existe desde o SAP R/3 4.6 — e com o ABAP 7.4+, a SAP passou a recomendar fortemente o estilo OO em detrimento de FORMs. Entender classes e interfaces é obrigatório para trabalhar com RAP, CDS, BAdIs e qualquer extensão moderna do SAP.

## O que você vai aprender

| Conceito | Por que importa |
|----------|----------------|
| CLASS / METHOD | Substitui FORM — mais testável, encapsulado |
| Herança | Reutilização sem cópia de código |
| Interfaces | Contratos — base para BAdIs e exitpoints |
| TRY/CATCH | Tratamento de erros moderno |
