## 🧠 Como este repositório funciona

Cada aula segue o mesmo formato:

```
📖 Conceito     → explicação objetiva, sem papo de manual SAP
💻 Código       → exemplos reais, comentados linha a linha  
⚠️ Pegadinhas   → erros comuns que todo dev ABAP já cometeu
🏋️ Exercício    → pratique você mesmo
```

---

## ⚙️ Pré-requisitos

- Acesso a um sistema SAP (veja o [módulo 00](./00-setup/) para opções gratuitas)
- Vontade de entender o **porquê**, não só o **como**
- Café ☕ (opcional, mas recomendado)

---

## 🚀 Por onde começar?

1. Leia o [Setup do Ambiente](./00-setup/README.md)
2. Siga os módulos em ordem — cada um assume o anterior
3. Faça os exercícios. Sério.
