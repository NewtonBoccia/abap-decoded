# 04 — Banco de Dados & CDS

> ABAP acessa o banco de dados via **Open SQL** — uma camada de abstração que funciona em qualquer banco suportado pelo SAP (HANA, Oracle, MSSQL). Com o HANA, surgiram as **CDS Views** (Core Data Services), que movem a lógica de dados para o banco e são a base do RAP e do Fiori.

## O que você vai aprender

| Conceito | Por que importa |
|----------|----------------|
| SELECT / INSERT / UPDATE / DELETE | Operações básicas de banco |
| JOINs | Cruzar tabelas sem carregar tudo em memória |
| FOR ALL ENTRIES | Busca baseada em tabela interna |
| CDS Views | Modelo moderno — base do Fiori e RAP |
