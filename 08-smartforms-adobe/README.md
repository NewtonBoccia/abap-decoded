# 08 — Smartforms & Adobe Forms

> Smartforms e Adobe Forms são as formas de gerar documentos impressos (PDF) no SAP: notas fiscais, pedidos, etiquetas, contratos. Smartforms é a tecnologia mais antiga (SE71) — Adobe Forms (SFP) é o padrão moderno para sistemas novos.

## Aulas

- [01 — Smartforms Básico](./01-smartforms-basico.md)
- [02 — Adobe Forms](./02-adobe-forms.md)

## O que você vai aprender

| Conceito | Transação | Por que importa |
|----------|-----------|----------------|
| Estrutura do Smartform | SE71 | Entender nodes e layouts |
| Chamar Smartform via ABAP | SE38 | Gerar PDF programaticamente |
| Interface do Adobe Form | SFP | Passar dados ao formulário |
| Saída de formulários | NACE | Configurar output em processos SAP |
