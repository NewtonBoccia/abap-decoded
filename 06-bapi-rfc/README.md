# 06 — BAPI & RFC

> BAPIs (Business Application Programming Interfaces) são Function Modules estáveis, documentados e certificados pela SAP para criar, ler e modificar objetos de negócio (pedidos, materiais, clientes). RFC (Remote Function Call) é o protocolo de comunicação que permite chamar FMs entre sistemas SAP ou de sistemas externos.
