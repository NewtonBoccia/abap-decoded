# 05 — IDocs & EDI

> Este módulo é o coração da integração SAP. IDoc (Intermediate Document) é o formato nativo do SAP para troca de dados com sistemas externos, EDI e interfaces B2B. Entender IDocs é obrigatório para qualquer Integration Specialist SAP.
